# POC Whitelabe

This is a template for a Vite-powered React project with some additional features like:

- [Tailwind CSS](https://tailwindcss.com/) for easy styling

## Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm (version 7 or higher)

### Installation

1. Install dependencies: `npm install`
2. Start the development server: `npm run dev`

### Usage

To start the development server, run `npm run dev`. This will launch the app in development mode with hot reloading enabled. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

To build the app for production, run `npm run build`. This will create a `dist` folder with optimized production-ready code.
